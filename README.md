# OIBSIP_WebDevelopment_Task4_PizzaApp

##  Project Title
Full Stack Pizza Ordering Web Application

##  Objective
The objective of this project is to build a full stack pizza ordering web application where users can register, login, customize pizzas, and place orders. The project demonstrates frontend and backend integration with authentication and database connectivity.

##  Features
- User Registration and Login System
- Admin Login
- Email Verification / Authentication
- Forgot Password System
- Pizza Customization (Base, Sauce, Cheese, Veggies)
- Dashboard to View Pizza Varieties
- Full Stack Integration

##  Technologies Used
Frontend:
- React
- TypeScript
- Vite
- CSS

Backend:
- Node.js
- Express.js
- MongoDB

Other Tools:
- Git & GitHub
- JWT Authentication

##  Steps Performed
1. Designed the frontend UI using React and Vite.
2. Created backend APIs using Node.js and Express.
3. Connected MongoDB database for storing user and order data.
4. Implemented authentication and authorization.
5. Integrated frontend and backend.
6. Tested complete application workflow.

##  How to Run the Project

1. Install dependencies:
   npm install
2. Add environment variables in .env file.
3. Run the project:
   npm run dev

##  Outcome
Successfully developed a working full stack pizza ordering web application with authentication and customization features.
